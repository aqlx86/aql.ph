# Personal Site

Personal portfolio website built with Next.js 15.

## Getting Started

### Prerequisites

- Node.js 18.18.0 or higher (using nvm recommended)

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
# Build static export
npm run build

# Preview production build
npm start
```

The static files will be generated in the `out/` directory.

## Project Structure

- `src/app/` - Next.js app router pages and layouts
- `public/` - Static assets (images, CSS, fonts)
- `archived/` - Original HTML/PHP files for reference

## Original Files

All original files have been moved to the `archived/` directory for future reference.
